## maven配置

```xml
<repositories>
   <repository>
      <id>chengzhimeow-maven-repo-releases</id>
      <name>橙汁喵の仓库</name>
      <url>https://maven.chengzhimeow.cn/releases</url>
   </repository>
</repositories>

<dependencies>
   <dependency>
      <groupId>cn.chengzhiya</groupId>
      <artifactId>MHDF-Reflection</artifactId>
      <version>1.0.0</version>
      <scope>compile</scope>
   </dependency>
</dependencies>
```
